Playto Payout Engine

Cross-border payment infrastructure for Indian businesses

API Endpoints

GET /api/v1/merchants
GET /api/v1/merchants/:id/balance
POST /api/v1/payouts
GET /api/v1/payouts?merchant_id=:id
POST /api/seed

Features implemented:

• Money integrity: BigInteger in paise, database-level SQL aggregation
• Concurrency: select_for_update() with row-level locking
• Idempotency: Keys scoped per merchant, 24h expiry
• State machine: Enforced in Payout.save() - blocks illegal transitions
• Retry logic: 30s timeout, exponential backoff, max 3 attempts
